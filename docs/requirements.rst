Requirements
============

The requirements necessary to use contribute to this Django Project are:

- **python3** and **pip3**
- **virtualenv and virtualenvwrapper**
- **Firefox** (selenium's webdriver in functional test)
- **GNU gettext** (to use Internationalization)
